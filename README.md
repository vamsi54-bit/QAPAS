Quiz Assessment System 🎯
A simple Python-based Quiz Assessment System built using Python, Pandas, and Matplotlib.
This project allows users to answer multiple-choice questions, calculates scores, analyzes performance, and displays results visually using graphs.

🧠 How It Works


Questions are stored in a list of dictionaries.


Questions are shuffled randomly.


User enters answers (A/B/C/D).


Program checks correctness.


Score and percentage are calculated.


Pandas creates a performance table.


Matplotlib generates a bar chart.


Old-school console projects like this are lowkey the best way to actually understand programming. No fancy frameworks. Just pure logic grinding. That’s where real coding instincts get built.
